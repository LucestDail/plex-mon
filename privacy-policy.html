Privacy Policy

Last Updated: November 7, 2024

English 한국어

Introduction

This Privacy Policy describes how "What's this" Chrome Extension ("we", "our", or "the extension") collects, uses, and protects your information.

Information We Collect

1. API Key

  • What: Your Google Gemini API key
  • How: Entered manually by you in the extension
  • Storage: Stored locally on your device only using Chrome's storage API
  • Purpose: To authenticate API requests for text summarization

2. Summary History

  • What: Your previous text summaries and timestamps
  • How: Automatically saved when you generate a summary
  • Storage: Stored locally on your device only (maximum 50 entries)
  • Purpose: To provide quick access to your summary history

3. Web Page Content

  • What: Text content from web pages you choose to summarize
  • How: Extracted only when you click "Summarize"
  • Processing: Sent directly to Google's Gemini API for summarization
  • Storage: NOT stored by our extension

How We Use Your Information

  • API Key: Used only to authenticate with Google's Gemini API
  • Summary History: Displayed in the extension's history tab for your convenience
  • Web Page Content: Sent to Gemini API only when you request a summary

Data Storage and Security

  • All data is stored locally on your device using Chrome's chrome.storage.local API
  • No data is sent to our servers (we don't have any servers)
  • Your API key is never shared with any third parties except Google's Gemini API
  • Web page content is only sent to Google's Gemini API when you explicitly request a summary

Third-Party Services

Google Gemini API

  • We use Google's Gemini API to generate text summaries
  • When you request a summary, the text content is sent to Google's servers
  • Google's privacy policy applies to this data: https://policies.google.com/privacy
  • We do not control how Google processes this data

Your Rights and Choices

You have the right to:

  • Delete your API key: Click the "Show/Hide" button and clear the input field
  • Delete summary history: Click the delete button on individual entries or clear all history
  • Stop using the extension: Uninstall the extension to remove all stored data

Data Deletion

To delete all your data:

  1. Open the extension side panel
  2. Delete your API key from the settings
  3. Clear all summary history
  4. Uninstall the extension from chrome://extensions/

All data will be permanently deleted from your device.

Children's Privacy

This extension is not intended for users under the age of 13. We do not knowingly collect information from children under 13.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by updating the "Last Updated" date at the top of this policy.

Contact Us

If you have any questions about this Privacy Policy, please contact us:

  • Email: lucestdail@kakao.com
  • GitHub: https://github.com/lucestdail

소개

본 개인정보처리방침은 "뭔 내용이지" Chrome 확장 프로그램("본 확장 프로그램")이 귀하의 정보를 어떻게 수집, 사용 및 보호하는지 설명합니다.

수집하는 정보

1. API 키

  • 내용: Google Gemini API 키
  • 수집 방법: 사용자가 확장 프로그램에 직접 입력
  • 저장 위치: Chrome의 storage API를 사용하여 귀하의 기기에만 로컬로 저장
  • 목적: 텍스트 요약을 위한 API 요청 인증

2. 요약 이력

  • 내용: 이전 텍스트 요약 결과 및 타임스탬프
  • 수집 방법: 요약을 생성할 때 자동으로 저장
  • 저장 위치: 귀하의 기기에만 로컬로 저장 (최대 50개 항목)
  • 목적: 요약 이력에 빠르게 접근할 수 있도록 제공

3. 웹 페이지 콘텐츠

  • 내용: 요약하려는 웹 페이지의 텍스트 콘텐츠
  • 수집 방법: "요약하기" 버튼을 클릭할 때만 추출
  • 처리: Google의 Gemini API로 직접 전송되어 요약 처리
  • 저장: 본 확장 프로그램은 저장하지 않음

정보 사용 방법

  • API 키: Google의 Gemini API 인증에만 사용
  • 요약 이력: 확장 프로그램의 이력 탭에 표시하여 편의성 제공
  • 웹 페이지 콘텐츠: 사용자가 요약을 요청할 때만 Gemini API로 전송

데이터 저장 및 보안

  • 모든 데이터는 Chrome의 chrome.storage.local API를 사용하여 귀하의 기기에만 로컬로 저장됩니다
  • 자체 서버로 데이터를 전송하지 않습니다 (자체 서버가 없습니다)
  • API 키는 Google의 Gemini API를 제외한 제3자와 공유되지 않습니다
  • 웹 페이지 콘텐츠는 사용자가 명시적으로 요약을 요청할 때만 Google의 Gemini API로 전송됩니다

제3자 서비스

Google Gemini API

  • 텍스트 요약을 생성하기 위해 Google의 Gemini API를 사용합니다
  • 요약을 요청하면 텍스트 콘텐츠가 Google의 서버로 전송됩니다
  • 이 데이터에는 Google의 개인정보처리방침이 적용됩니다: https://policies.google.com/privacy?hl=ko
  • Google이 이 데이터를 처리하는 방법은 본 확장 프로그램이 통제하지 않습니다

귀하의 권리 및 선택

귀하는 다음과 같은 권리가 있습니다:

  • API 키 삭제: "표시/숨기기" 버튼을 클릭하고 입력 필드를 지웁니다
  • 요약 이력 삭제: 개별 항목의 삭제 버튼을 클릭하거나 전체 이력을 지웁니다
  • 확장 프로그램 사용 중지: 확장 프로그램을 제거하여 저장된 모든 데이터를 삭제합니다

데이터 삭제

모든 데이터를 삭제하려면:

  1. 확장 프로그램 사이드 패널 열기
  2. 설정에서 API 키 삭제
  3. 모든 요약 이력 지우기
  4. chrome://extensions/에서 확장 프로그램 제거

모든 데이터가 귀하의 기기에서 영구적으로 삭제됩니다.
