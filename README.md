# 뭔 내용이지 / What's this

웹 페이지나 텍스트를 AI로 간결하게 요약하는 Chrome 확장 프로그램입니다.

## 📋 프로젝트 개요

정보 과부하 시대에 긴 기사, 블로그 포스트, 문서 등을 빠르게 이해하고 핵심 내용을 파악할 수 있도록 도와주는 Chrome 확장 프로그램입니다. Google의 Gemini API를 활용하여 웹 페이지의 본문을 자동으로 추출하고, 핵심 내용을 간결하게 요약하여 제공합니다.

## ✨ 주요 기능

### 1. 자동 인식 모드 (Auto Detect)
- 현재 열려있는 웹 페이지의 본문을 자동으로 추출
- 광고, 네비게이션, 푸터 등 불필요한 요소를 제거하고 본문만 추출
- 2단계 처리 프로세스:
  - 1단계: Gemini API를 활용한 본문 전처리 (유의미한 내용만 추출)
  - 2단계: 전처리된 본문을 간결하게 요약

### 2. 직접 선택 모드 (Manual Select)
- 페이지에서 원하는 영역을 클릭하여 선택
- 여러 개의 영역을 선택 가능
- 선택한 영역만 요약하여 더 정확한 결과 제공

### 3. 직접 입력 모드 (Manual Input)
- 사용자가 원하는 텍스트를 직접 입력하여 요약
- 복사한 텍스트나 메모를 빠르게 요약

### 4. 사이드 패널 UI
- 페이지 전환 시에도 열려있는 사이드 패널
- 요약 결과를 계속 확인 가능
- 깔끔하고 직관적인 인터페이스

### 5. 다국어 지원 (i18n)
- 브라우저 언어 설정에 따라 자동으로 한국어/영어 전환
- 요약 결과도 선택한 언어로 제공

### 6. 요약 이력 관리
- 요약 결과를 자동으로 저장
- 이전 요약 결과를 쉽게 확인하고 복사 가능
- 최대 50개의 요약 이력 저장

### 7. Gemini API 통합
- Google Gemini 2.5 Flash 모델 활용
- API 키 자동 저장 및 관리
- API 키 표시/숨기기 토글 기능

## 🚀 설치 방법

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택

## 🔑 Gemini API 키 발급

1. [Google AI Studio](https://makersuite.google.com/app/apikey)에 접속
2. Google 계정으로 로그인
3. "Create API Key" 버튼 클릭
4. 생성된 API 키를 복사
5. 확장 프로그램 사이드 패널에서 API 키 입력 (한 번만 입력하면 자동 저장됩니다)

**참고**: Gemini API는 무료로 사용할 수 있습니다.

## 📖 사용 방법

1. 확장 프로그램 아이콘 클릭 (사이드 패널이 열립니다)
2. Gemini API 키 입력 (최초 1회만, 자동 저장됨)
3. 원하는 모드 선택:
   - **자동 인식**: "요약하기" 버튼 클릭
   - **직접 선택**: "선택 모드 시작" → 원하는 영역 클릭 → "선택 모드 종료" → "요약하기"
   - **직접 입력**: 텍스트 입력 후 "요약하기" 클릭
4. 요약 결과 확인
5. 필요시 "📚 History" 버튼으로 이전 요약 확인

## 🛠️ 기술 스택

- **Chrome Extension Manifest V3**: 최신 Chrome 확장 프로그램 표준
- **Google Gemini 2.5 Flash API**: AI 기반 텍스트 요약
- **Chrome Side Panel API**: 페이지 전환 시에도 열려있는 UI
- **Chrome i18n API**: 다국어 지원
- **Chrome Storage API**: 로컬 데이터 저장
- **Chrome Scripting API**: 동적 스크립트 주입

## 🎯 주요 특징

### 기술적 특징
- **동적 스크립트 주입**: Manifest V3의 `chrome.scripting.executeScript`를 활용하여 안정적인 텍스트 추출
- **이중 폴백 메커니즘**: 동적 주입 실패 시 content script 메시지 방식으로 자동 전환
- **로컬 스토리지 활용**: API 키와 요약 이력을 안전하게 저장
- **에러 처리**: 각 단계별 상세한 에러 메시지 제공
- **브라우저 언어 자동 감지**: 사용자 브라우저 설정에 따라 자동으로 언어 전환

### 사용자 경험
- **간결한 요약**: 핵심만 간단명료하게 요약 (200자 이내)
- **시각적 피드백**: 로딩 상태와 진행률 표시
- **요약 이력**: 이전 요약 결과를 쉽게 확인하고 복사
- **직관적인 UI**: 모드 전환과 기능 사용이 간편

## 📝 활용 시나리오

1. **뉴스 및 기사 읽기**
   - 긴 뉴스 기사를 빠르게 파악
   - 여러 기사를 비교하며 핵심 내용만 확인

2. **학습 및 연구**
   - 논문이나 연구 자료의 핵심 내용 파악
   - 학습 자료의 요점 정리

3. **업무 효율성 향상**
   - 보고서나 문서의 핵심 내용 확인
   - 회의 자료나 공지사항의 요약 확인

4. **콘텐츠 소비 최적화**
   - 블로그 포스트나 긴 글의 핵심 파악
   - 소셜 미디어나 포럼의 긴 게시글 요약

