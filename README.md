# 뭔 내용이지

웹 페이지나 텍스트를 요약하는 크롬 확장 프로그램입니다.

## 프로젝트 개요

### 배경

현대 인터넷 환경에서는 매일 수많은 정보와 콘텐츠가 생성되고 공유됩니다. 사용자들은 긴 기사, 블로그 포스트, 문서 등을 빠르게 이해하고 핵심 내용을 파악하기를 원합니다. 그러나 시간이 부족하거나 정보 과부하 상황에서 모든 내용을 읽고 이해하는 것은 쉽지 않습니다.

이러한 문제를 해결하기 위해, Google의 Gemini API를 활용한 크롬 확장 프로그램을 개발하였습니다. 이 확장 프로그램은 웹 페이지의 본문 내용을 자동으로 추출하고, 인공지능을 통해 핵심 내용을 3줄로 요약하여 제공함으로써 사용자의 정보 소비 효율성을 높이고자 합니다.

### 주요 기능

본 프로젝트는 다음과 같은 핵심 기능을 제공합니다:

1. **직접 입력 모드**
   - 사용자가 원하는 텍스트를 직접 입력하여 요약
   - 복사한 텍스트나 메모를 빠르게 요약하고 싶을 때 활용

2. **자동 인식 모드**
   - 현재 열려있는 웹 페이지의 본문을 자동으로 추출
   - 광고, 네비게이션, 푸터 등 불필요한 요소를 제거하고 본문만 추출
   - 2단계 처리 프로세스:
     - 1단계: Gemini API를 활용한 본문 전처리 (유의미한 내용만 추출)
     - 2단계: 전처리된 본문을 3줄로 요약

3. **Gemini API 통합**
   - Google Gemini 2.5 Flash 모델 활용
   - 무료 API 제공으로 접근성 확보
   - API 키 자동 저장 및 관리 기능

4. **사용자 친화적 인터페이스**
   - 직관적인 모드 전환 (직접 입력 / 자동 인식)
   - 실시간 로딩 상태 표시
   - API 키 표시/숨기기 토글 기능

### 기능 활용

#### 활용 시나리오

1. **뉴스 및 기사 읽기**
   - 긴 뉴스 기사를 빠르게 파악하고 싶을 때
   - 여러 기사를 비교하며 핵심 내용만 확인하고 싶을 때

2. **학습 및 연구**
   - 논문이나 연구 자료의 핵심 내용을 빠르게 파악
   - 학습 자료의 요점 정리

3. **업무 효율성 향상**
   - 보고서나 문서의 핵심 내용을 빠르게 확인
   - 회의 자료나 공지사항의 요약 확인

4. **콘텐츠 소비 최적화**
   - 블로그 포스트나 긴 글의 핵심 파악
   - 소셜 미디어나 포럼의 긴 게시글 요약

#### 기술적 특징

- **동적 스크립트 주입**: Manifest V3의 `chrome.scripting.executeScript`를 활용하여 안정적인 텍스트 추출
- **이중 폴백 메커니즘**: 동적 주입 실패 시 content script 메시지 방식으로 자동 전환
- **로컬 스토리지 활용**: API 키를 안전하게 저장하여 사용자 편의성 향상
- **에러 처리**: 각 단계별 상세한 에러 메시지 제공

### 결과

본 프로젝트를 통해 다음과 같은 성과를 달성하였습니다:

1. **사용자 경험 개선**
   - 웹 페이지의 핵심 내용을 3줄로 요약하여 정보 소비 시간 단축
   - 간단한 클릭만으로 요약 기능 제공으로 사용 편의성 향상
   - 두 가지 모드 제공으로 다양한 사용 시나리오 지원

2. **기술적 성과**
   - Google Gemini API를 활용한 효율적인 텍스트 요약 구현
   - 웹 페이지 본문 자동 추출 및 전처리 기능 구현
   - Chrome Extension Manifest V3 표준 준수

3. **확장 가능성**
   - 다양한 LLM API로 교체 가능한 구조
   - 추가 기능 확장이 용이한 모듈화된 코드 구조
   - 사용자 설정 및 커스터마이징 가능한 구조

4. **실용성**
   - 무료 API 활용으로 비용 부담 없음
   - 설치 및 사용이 간편한 크롬 확장 프로그램 형태
   - 다양한 웹 페이지에서 동작하는 범용성

본 프로젝트는 정보 과부하 시대에 사용자들이 효율적으로 정보를 소비할 수 있도록 돕는 실용적인 도구로, 지속적인 개선과 확장을 통해 더욱 유용한 서비스로 발전시킬 수 있을 것으로 기대됩니다.

## 기능

- **직접 입력 모드**: 텍스트를 직접 입력하여 요약
- **자동 인식 모드**: 현재 웹 페이지의 본문 텍스트를 자동으로 추출하여 요약
- **Gemini API 연동**: Google Gemini API를 사용하여 3줄 요약 생성

## 설치 방법

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택

## 아이콘 설정

`icons` 폴더에 다음 크기의 아이콘 이미지를 추가하세요:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

아이콘이 없어도 확장 프로그램은 정상 작동합니다.

## Gemini API 키 발급

1. [Google AI Studio](https://makersuite.google.com/app/apikey)에 접속
2. Google 계정으로 로그인
3. "Create API Key" 버튼 클릭
4. 생성된 API 키를 복사
5. 확장 프로그램 팝업에서 API 키 입력 (한 번만 입력하면 자동 저장됩니다)

**참고**: Gemini API는 무료로 사용할 수 있습니다.

## 사용 방법

1. 확장 프로그램 아이콘 클릭
2. Gemini API 키 입력 (최초 1회만, 자동 저장됨)
3. "직접 입력" 또는 "자동 인식" 모드 선택
4. 직접 입력 모드: 텍스트 입력 후 "요약하기" 클릭
5. 자동 인식 모드: "자동 요약하기" 클릭
6. 결과 확인
